# Exercise 1: Setting Up JUnit

## Problem Statement
Configure JUnit 5 with Maven for TDD exercises.

## Approach
Maven project with junit-jupiter and maven-surefire-plugin for automated test execution.

## How to Run
cd Week-1/JUnit-Mockito-SLF4J
mvn test -Dtest=JUnitSetupTest
