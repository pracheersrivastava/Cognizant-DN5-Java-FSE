# Mockito Exercise 1: Mocking and Stubbing

## Problem Statement
Mock external API dependency and stub return values.

## Approach
Mockito @Mock and @InjectMocks with when().thenReturn() stubbing.

## How to Run
cd Week-1/JUnit-Mockito-SLF4J
mvn test -Dtest=MockingAndStubbingTest

## Expected Output
```
Service returns stubbed Cognizant data from mock
```
