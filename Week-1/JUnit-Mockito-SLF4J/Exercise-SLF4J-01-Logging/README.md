# SLF4J Exercise 1: Error and Warning Logging

## Problem Statement
Configure SLF4J with Logback and emit ERROR/WARN log levels.

## Approach
LoggerFactory facade with Logback implementation; structured console logging.

## How to Run
cd Week-1/JUnit-Mockito-SLF4J/Exercise-SLF4J-01-Logging
mvn compile exec:java

## Expected Output
```
ERROR and WARN log lines printed with timestamps and logger names
```
