# Exercise 2: E-commerce Platform Search

## Problem Statement
Compare linear and binary search for product lookup in an e-commerce catalog.

## Approach
Linear search O(n); binary search O(log n) on sorted Product array using Comparable.

## How to Run
cd "Week-1/Data Structures and Algorithms/Exercise-02-Ecommerce-Platform-Search"
javac -d out src/com/cognizant/javadfse/week1/dsa/search/*.java
java -cp out com.cognizant.javadfse.week1.dsa.search.EcommerceSearchExample

## Expected Output
```
Linear and binary search both find product ID 107 after sorting inventory.
```
