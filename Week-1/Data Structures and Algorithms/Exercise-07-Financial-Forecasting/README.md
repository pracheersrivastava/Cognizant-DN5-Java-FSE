# Exercise 7: Financial Forecasting

## Problem Statement
Predict future financial values using recursive algorithms based on growth rates.

## Approach
Recursive compound growth with memoized fast exponentiation; naive recursion O(N), optimized O(log N).

## How to Run
cd "Week-1/Data Structures and Algorithms/Exercise-07-Financial-Forecasting"
javac -d out src/com/cognizant/javadfse/week1/dsa/forecasting/*.java
java -cp out com.cognizant.javadfse.week1.dsa.forecasting.ForecastingExample

## Expected Output
```
Projected future values for multiple periods printed to console.
```
