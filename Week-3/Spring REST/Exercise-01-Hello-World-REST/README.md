# Hello World RESTful Web Service

## Problem Statement
Create a Spring Boot 3 Maven project exposing a basic REST endpoint.

## Approach
Spring Boot starter-web with a single @RestController mapped to GET /hello.

## How to Run
cd "Week-3/Spring REST/Exercise-01-Hello-World-REST"
mvn spring-boot:run
curl http://localhost:8080/hello

## Expected Output
```
Hello World from Spring Boot 3 REST Service
```
